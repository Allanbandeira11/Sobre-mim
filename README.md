### 📂 Projetos de Impacto

#### 📱 ALLIX - Painel de Atendimento Inteligente
O projeto **ALLIX** é um ecossistema SaaS robusto para centralização de atendimentos WhatsApp.
* **Core:** React Native + Supabase Realtime.
* **Funcionalidades:** Fila única sincronizada, sistema de status dinâmico (Online/Pausa/Offline) e painel administrativo para controle de unidades.
* **Parceria:** Desenvolvido em colaboração estratégica com o desenvolvedor **Antonio**.

#### 🎮 Vida Nova RP - Servidor SAMP
Fundador e desenvolvedor líder de um dos servidores de Roleplay mais resilientes da cena SAMP.
* **Sistemas:** Criação de scripts do zero para economia, empregos e interações sociais.
* **Performance:** Otimização para centenas de players simultâneos em ambiente de baixa latência.
* **Gestão:** Administração de comunidade e infraestrutura de servidores Linux.
